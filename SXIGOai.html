✦ SXIGOai Help

Premium AI Assistant with Web Search & Galaxy Theme

💬 Chat

Type your message and press Enter or click ↑. Use Shift+Enter for new lines. Select a model from the model bar. Adjust temperature, top_p, and max_tokens in Settings.

🌐 Web Search & Crawl

Click the 🌐 button in the header to open Web Tools. Switch between Search (DuckDuckGo) and Crawl (fetch any URL). Results can be sent directly to your chat input. Search history is saved locally.

🎨 Themes

12 themes available including the immersive Galaxy theme with dynamic stars, shooting stars, aurora waves, nebula orbs, and cosmic rings. Open Settings → Theme to browse.

⌨ Keyboard Shortcuts

  • ⌘N New Chat
  • ⌘⇧F Search Messages
  • ⌘⇧W Web Search
  • ⌘B Sidebar
  • ⌘, Settings
  • ⌘⇧P Playground
  • ⌘⇧C Clear Chat
  • Esc Close panels / Stop

🕸 Web Crawl

Enter any URL and click Go to fetch and extract its text content. Useful for getting article text, documentation, or any web page content directly in the app.

💡 Tip: Use galaxy theme for the best visual experience. Switch between dark and light modes automatically based on your system preference.
